# My AI in Excel Sheet Projects


An AI-powered Google Sheets project designed to automate and enhance data processing, analysis, and content generation workflows. This repository integrates intelligent features for dataset cleaning, transforming raw data into structured table formats, sentiment analysis of consumer feedback, dataset summarisation and interpretation, dashboard and chart generation for visual insights, automated product description creation, translation across languages, and other AI-assisted analytical tasks. The project aims to improve efficiency in data management, business intelligence, and decision-making through seamless AI integration within Google Sheets.
